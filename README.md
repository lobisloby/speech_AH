## ✨ Features

- 🎙️ **Text-to-Speech** — Generate natural speech with controllable creativity and expression
- 🧬 **Zero-shot Voice Cloning** — Clone voices from a short audio sample (no training required)
- 🎧 **20 Built-in Voices** — Curated voices across multiple styles and locales
- 📊 **Waveform Player** — Interactive audio playback with WaveSurfer.js
- 🏢 **Multi-Tenant Support** — Team-based access via Clerk Organizations
- 💳 **Usage-Based Billing** — Metered billing powered by Polar
- 📚 **Generation History** — Revisit and reuse previous outputs
- 📱 **Fully Responsive UI** — Optimized for mobile and desktop

## 🚀 Getting Started

### Requirements

- Node.js 20.9+
- PostgreSQL database (Prisma-compatible)
- Clerk account (Organizations enabled)
- Cloudflare R2 bucket
- Modal account (GPU inference)
- Polar account (billing)

---

### 1. Clone the repository


git clone [https://github.com/AyoubCoder/neuralvoice.git](https://github.com/lobisloby/speech_AH.git)
cd neuralvoice
npm install
